# Portafolio Profesional - Salomon Moreyra Peralta

Este es mi portafolio profesional desarrollado con HTML, CSS y JavaScript. Presenta un diseño moderno, responsivo y con efectos de alta tecnología.

## Características
- **Fondo de Video**: Fondo cinemático de alta tecnología.
- **Animaciones de Constelaciones**: Efectos interactivos con el mouse.
- **Terminal Hacker**: Modal interactivo para visualizar habilidades técnicas.
- **Diseño Responsivo**: Adaptado para móviles, tablets y escritorio.
- **Contacto**: Enlaces directos a WhatsApp, Email y Redes Sociales.

## Tecnologías Utilizadas
- HTML5 Semántico
- CSS3 (Glassmorphism, Flexbox, Grid)
- JavaScript Vanilla (Animaciones y lógica de UI)
- Font Awesome para Iconografía

## Estructura del Proyecto
- `index.html`: Estructura principal.
- `style.css`: Estilos, animaciones y diseño visual.
- `script.js`: Lógica de interactividad y efectos.
- `assets/`: Carpeta de recursos (imágenes, iconos).
